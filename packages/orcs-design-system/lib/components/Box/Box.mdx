## Properties

You can use `space`, `layout`, `border`, and `color` properties with `Box`.

### For reference, see:

- [space on Styled-System](https://styled-system.com/table#space)
- [layout on Styled-System](https://styled-system.com/table#layout)
- [border on Styled-System](https://styled-system.com/table#border)
- [color on Styled-System](https://styled-system.com/table#color)
- [systemtheme.js](https://github.com/orchestrated-io/orca-design-system/blob/master/packages/orcs-design-system/lib/systemtheme.js) (our arrays and scales for design system values)

By default, we've set certain values on `Box` properties as documented in the table below.

<Props of={Box} />

**Note:** If specific `margin` or `padding` properties fail to override the defaults, specify the overall property and then the specific property. For example, to set a `Box` with a `margin-bottom: 8px`, use

`<Box m={0} mb={3}>`

## Upgrading

This component is a **breaking change** that replaces the [deprecated Layout component](https://styleguide.orchestrated.io/#layout). Use the following table as a guide to convert from `Layout` properties to `Box` properties. Refer to [systemtheme.js](https://github.com/orchestrated-io/orca-design-system/blob/master/packages/orcs-design-system/lib/systemtheme.js) for values.

| **Layout**        | **Box**             |
| ----------------- | ------------------- |
| `width`           | `width`             |
| `height`          | `height`            |
| `padding`         | `p={3}`             |
| `paddingDouble`   | `p={5}`             |
| `paddingHalf`     | `p={2}`             |
| `background`      | `bg="greyLightest"` |
| `backgroundDark`  | `bg="greyDark"`     |
| `backgroundWhite` | `bg="white"`        |
| `roundedCorners`  | `borderRadius={2}`  |
