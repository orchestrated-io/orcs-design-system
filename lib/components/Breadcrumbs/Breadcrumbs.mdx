# Breadcrumbs

[Default](#default) | [White variant](#white-variant) | [Properties](#properties)

Breadcrumbs help users understand their location within a hierarchy and provide easy navigation back to previous levels.

The Breadcrumbs component accepts a `config` array of `{ label, to }` objects. The last item is rendered as the current page (bold, not a link).
