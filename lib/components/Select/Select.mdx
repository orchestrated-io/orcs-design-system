# Select

[Default](#default) | [Disabled](#disabled) | [Multi-select](#multi-select) | [Inverted](#inverted) | [With default value](#with-selected-defaultvalue) | [With creatable](#with-creatable) | [With async](#with-async) | [With grouped options](#with-grouped-options) | [Properties](#properties)

This component uses React Select: https://react-select.com/home. See link for all uses and props.

Select values are defined as separate objects. for example:

```js
const options = [
  { value: "caramel", label: "Caramel" },
  { value: "chocolate", label: "Chocolate" },
  { value: "strawberry", label: "Strawberry" },
  { value: "vanilla", label: "Vanilla" }
];
```
