## Usage

The typography system requires you to import each subcomponent you want to use, for example:

```
import { H1, H2, H3, H4, H5, H6, P, Text, Quote, Small, Code } from "orcs-design-system";
```

## Font families

By default, the design system uses **Open Sans**: https://fonts.google.com/specimen/Open+Sans with other sans serif families as fallback. The default font weight is 400, and the system also uses 600 for bold or emphasised text and 300 for light weights (to be used sparingly).

Font defaults can be changed via [systemtheme.js](https://github.com/orchestrated-io/orcs-design-system/blob/master/packages/orcs-design-system/lib/systemtheme.js).

<Typeset
  fontSizes={[
    themeGet("fontSizes.0"),
    themeGet("fontSizes.1"),
    themeGet("fontSizes.2"),
    themeGet("fontSizes.3"),
    themeGet("fontSizes.4"),
    themeGet("fontSizes.5"),

    themeGet("fontSizes.7"),
    themeGet("fontSizes.8"),
    themeGet("fontSizes.9")
  ]}
  fontFamily={themeGet("font")}
/>

## Text styling

### Size and emphasis

To render type smaller or larger, use `sizing="small"` or `sizing="large"` on a heading, or if needed you can use `fontSize` property or the `Small` element as demonstrated below. Unless it is for a very specific purpose, strongly recommend minimising the resizing of text and simply using the appropriate type element the next size up (or down) on the design system scale.

<Canvas>
  <Story name="Styling/Sizing">
    <Spacer my={3}>
      <H5 sizing="small">Small variant of H5</H5>
      <H5 sizing="large">Large variant of H5</H5>
      <H5 fontSize={2}>
        Even smaller H5, scaled down two steps in the fontSize array
      </H5>
      <P fontSize="10px">Very small text using a specific fontSize value</P>
      <Small>Small text using the Small component</Small>
    </Spacer>
  </Story>
</Canvas>

To render type with emphasis, use `weight="bold"` or wrap the type in a `<strong>` tag.

<Canvas>
  <Story name="Styling/Emphasis">
    <Spacer my={3}>
      <P weight="bold">Bold paragraph text lorem ipsum dolor sit amet</P>
      <P>
        Only <strong>partially bolded</strong> paragraph
      </P>
    </Spacer>
  </Story>
</Canvas>

### Colours

To apply colour to text, use the `color` property in the `Text` component. Usage should be limited to the colours in the example below except in very specific design cases.
