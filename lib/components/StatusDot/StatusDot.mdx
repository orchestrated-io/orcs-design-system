# StatusDot

[Default StatusDot](#default-StatusDot) | [Colour](#colour) | [With icon](#with-icon) | [Properties](#properties)

Status dot component is used to specify a notification or status on an item. For example, this can be used to represent a true/false, met/not met, on/off or new/received/updated states. One, or multiple can be used to represent different things.

The horizontal layout and tooltip functionality comes from other components (Flex and Popover respectively) see example code for use.
